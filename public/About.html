About ForexCompass

This web application is designed to assist users in managing their forex trading strategies. Built using JavaScript, React, and Firebase for authentication, ForexCompass provides a comprehensive platform for real-time forex data, strategy creation, and market analytics.

Features

  • Real-Time Forex Data: Access live forex pair rates to make informed trading decisions with up-to-the-minute information.
  • Strategy Creation: Create and customize trading strategies through a user-friendly form. The application generates MetaTrader-compatible templates incorporating all necessary indicators and parameters.
  • Market News: Stay informed with an aggregated news section that gathers relevant forex market news from multiple sources.
  • User Management: Secure authentication for logging in, registering, and managing user profiles and settings.
  • Analytics: Learn about indicators used in MetaTrader and how to create a template to import to help with analysis.
